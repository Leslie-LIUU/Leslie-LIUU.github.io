在用好用的工具向软件

2023/12 # Sharing

粗略整理一些体验不错长期使用的偏工具属性的软件，基于 Windows 平台，包含个人使用体验、官方介绍（部分由 ChatGPT 写）以及官方网站。有汉化版需要的可以去🔗找或者邮件我。

影音类

音乐标签

  • 修改本地音乐标签的无敌存在，可以联机查找多个源的封面、歌词……手动匹配，也可以全自动再手动检查。👍作者还有一个 Android 的本地音乐播放器也非常好用，长期使用，还可以联动 Android 版的音乐标签。
  • 编辑歌曲的标题，专辑，艺术家等信息的应用程序，支持 FLAC, APE, WAV, AIFF, WV, TTA, MP3, MP4, M4A, OGG, MPC, OPUS, WMA, DSF, DFF 等音频格式，绿色无广告，无任何功能限制。

https://www.cnblogs.com/vinlxc/

Foobar2000

  • 通常用来检查和管理本地音乐、配合 cue 文件分割音轨、转换 flac、wav 格式。
  • 音频播放器，支持多种音频格式，包括 MP3、AAC、WMA、FLAC 等。它的界面简洁，功能强大，支持音频播放、转换、标签编辑等多种功能。foobar2000 还支持插件扩展，用户可以根据自己的需求安装不同的插件来增强软件的功能。

https://www.foobar2000.org/

PotPlayer

  • 功能齐全、支持插件、稳定，本地视频播放器首选。
  • 多媒体播放器，它支持大多数音频和视频格式，并具有许多高级功能，例如屏幕截图，字幕支持，音频和视频滤镜等。

https://potplayer.daum.net/?lang=zh_CN

VLC

  • 本地视频播放器，还有 MPC-BE，主要是作为 Pot 的备用。
  • 自由、开源的跨平台多媒体播放器及框架，可播放大多数多媒体文件，以及 DVD、音频 CD、VCD 及各类流媒体协议。

https://www.videolan.org/

haneWIN NFS Server

  • 简单的实现NFS共享服务，用来和投影仪、电视的 kodi 实现联动，传输速度和稳定性都还不错，个人体验比 FTP、SMB 好。
  • NFS 服务器软件，可让 Windows 主机作为 NFS 服务器共享文件和目录。它支持 NFSv3 和 NFSv4 协议，并提供安全的身份验证和访问控制功能。

https://www.hanewin.net/index.html

MKVToolNix

  • MKV 的兼容性非常好，基本上是用来增改音轨、字幕轨，无损输出且封装效率高。
  • 开源 MKV 格式多媒体文件处理工具，支持创建、修改、分离、合并和检查 MKV 文件，可用于添加字幕、音轨、章节信息等操作。

https://mkvtoolnix.download/

哔哩下载姬Downkyi

  • 相对在线转换的网站要稳定和高效很多。
  • 哔哩哔哩网站视频下载工具，支持批量下载，支持 8K、HDR、杜比视界，提供工具箱（音视频提取、去水印等）。

https://github.com/leiurayer/downkyi

小功能类

Open Hardware Monitor

  • 主要是用来监测温度（前段时间温控芯片出问题了🙄）
  • 免费开源的硬件信息监控工具，可监测 CPU、GPU、硬盘、温度、电压等硬件参数，并提供实时报告和图表分析。适用于 Windows 和 Linux 平台。

https://openhardwaremonitor.org/

Traffic Monitor

  • 固定在任务栏一角的网速、资源监控器，美观好用，稳定，还可以定制快捷功能。
  • 网速监控悬浮窗软件，可以显示当前网速、CPU 及内存利用率，支持嵌入到任务栏显示，支持更换皮肤、历史流量统计等功能。

https://github.com/zhongyang219/TrafficMonitor

Digital Clock 4

  • 固定的悬浮时间，通过设置样式，存在感低但是想看时间的时候可以一眼看到。
  • 桌面数字时钟软件，提供多种样式和设置选项，支持显示当前时间和日期。

https://sourceforge.net/projects/digitalclock4/

工具类

SIV - System Information Viewer

  • 功能导向
  • 系统信息查看器，显示诸如 CPU 温度、风扇转速、内存使用情况、网络带宽等系统信息。还可以显示一些高级信息，例如 CPU 型号、主板型号、BIOS 版本等。此外，SIV 还具有一些系统调整和优化功能，例如修改 CPU 电压、设置风扇曲线等。

http://rh-software.com/

SpaceSniffer

  • 用图块的方式看文件的占用情况、路径，整理文件的时候可以用。
  • 磁盘空间占用分析工具，可帮助用户可视化地查看磁盘上的文件和文件夹占用情况，方便用户管理和清理磁盘空间。

http://www.uderzo.it/

7-zip

  • 体验最好的解压缩软件。
  • 文件压缩和解压缩软件，支持多种格式，包括 7z、ZIP、RAR、GZIP、BZIP2 等。它具有高压缩比和快速压缩和解压缩速度的优点，可用于文件备份、文件传输和节省磁盘空间。7-Zip 还提供了命令行版本和图形化用户界面版本。

https://www.7-zip.org/

Cheat Engine

  • 用来修改软件的时间进程的（可以用在百度云上🤫），并没有用在游戏上。
  • 内存编辑器和调试器，可用于修改游戏和应用程序中的内存数据，例如改变分数、生命值等。它还提供了脚本编辑和代码注入功能，适用于 Windows、Linux、macOS 等平台。

https://cheatengine.org/

Bitcomet

  • BT软件很多，综合使用下来推荐这个，uTorrent 也不错。
  • 免费的 P2P 文件下载工具，支持 HTTP、FTP、BT 等多种下载协议，具有高速下载、任务调度、BT 健康检查等功能。

https://www.bitcomet.com/en

Snipaste

  • 第三方截图软件。
  • 易用且功能强大的截图工具，可以快速截取屏幕上的任意区域，并支持简单的编辑和标注。用户可以自定义快捷键来方便截图，同时也支持多个显示器和在它们之间拖动截图。

https://zh.snipaste.com/
